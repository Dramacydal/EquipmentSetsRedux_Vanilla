EquipmentSetsRedux_Vanilla
=====================

This is a revamped [EquipmentSets addon](https://www.curseforge.com/wow/addons/equipmentsets) for better Vanilla support.

* Possibility to store unlimited amount of named equipment sets
* Equip, unequip, rename and remove equipment sets via added dropdown at character frame (or by commands and macroses)
* Properly handle trinkets, ammos, and dual wield weapons
* It is possible to equip sets while visiting bank
* Missing and currently equipped items are displayed in equipment set tooltip
* If an item belongs to an equipment set - it is now displayed in item's tooltip


List of available commands:
```
/es save <name> or <setId> <name> - Save set with with specific name or to specific slot  
/es unequip - Unequip all items  
/es remove <setId> - Remove equipment set  
/es load <setId> or <name> - Equip set by number or name  
/es list - List stored equipment sets  
/es equip <setId> or <name> - Equip set by number or name  
/es setposition <setId> <positionId> - Remove specific position (slotId) from stored set  
/es rename <setId> <name> - Set name for equipment set  
/es reset - Remove all equipment sets
```
